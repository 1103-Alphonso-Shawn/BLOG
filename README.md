# BLOG

# Flask Blog Project
simple blog web application built using Flask + SQLite.
Users can register, log in, create posts, and delete their own posts.

# Features
User registration and login system
Session-based authentication
Create blog posts
Delete posts
View all posts on dashboard
SQLite database integration

# TECH USED
Python
Flask
SQLite
HTML / CSS
Jinja2 templates
